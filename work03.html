Music Taste Comparator

| Live Site | Github |

A JavaScript project utilizing Spotify API.

Music Taste Comparator was built using JavaScript, Spotify API, Ajax, HTML5, CSS3, and D3.js

Features

User Login Users can log in using an existing Spotify account through the Spotify API. A demo account is provided for non-Spotify users.

User Statistics: Upon log in, user's Spotify data is fectched and displayed showing their top song and top genres.

Friend Search: User can search a friend's username to view their top songs of 2019 playlist. The searched users top songs and genres are incorporated into the display.

Recommended Tracks: Based on data from the current user and the searched user, a list of recommended tracks will be genarated
