O-O-T-D

Outfit of the Day

| Live Site | Github |

A MERN stack outfit organizer and generator.

O-O-T-D was built using MongoDB, Express, React, Node.js

Features

User Authentication Secure user authenication implented via bcrypt.js. Demo User account available for guests.

Items : User can create a new item with image upload through AWS. User can filter items based on category.

Collections: User can create a new collection consisting of items in their inventory. User can view index of all their current collections.

Generate: User can view collections based on filters.
